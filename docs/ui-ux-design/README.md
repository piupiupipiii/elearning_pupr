## Flow

1. (Landing)[./Judul.png]. When button "Mulai" clicked, it will redirect to Sign in page
2. Sign up / Sign in
3. (Welcome)[./Intro.png]. When button "Mulai Pembelajaran" clicked, it will redirect to Home/Modules (Beranda Modul)[./Beranda Modul.png] page
4. Default (Home/Modules page)[./Beranda Modul.png]. Highlighted button will change color to yellow, see (Home/Modules with highlighted button)[./Beranda Modul-1.png] for example.
5. When user click "Kompetensi Dasar", it will redirect to (Pendahuluan Kompetensi Dasar)[./Pendahuluan/KD.png]. The only option on this page is to back to Home/Modules page.
6. When user click "Video Materi", it will redirect to (Materials page)[./sub menu materi .3.png]. In this page, there would be button "Mulai" to start current unlocked material and several material card to choose and their own separate progress status (locked, unlocked, or done). I expect user can revisit finished materials too.
7. When user choose start material, it will redirect to (Material page)[./opsi video 1.1.png]. In this page, there would be a video on the right side and options to back to Materials selection page and continue to quiz.
8. On (quiz page)[KUIS 2.png], there would be a quiz and answer options (A/B/C/D). When user choose the right answer, there would be (checkmark icon pop-up)[./jawaban kuis benar-2.1.png]. When user choose the wrong answer, there would be (cross icon pop-up)[./jawaban kuis salah-2.1.png] and the page background will be chanegd to orange.
9. Back in the Home/Modules page, when user click on the "Media Pendukung", it will redirect to (Media Pendukung page)[./Media Pendukung.png].
